# 基础绘制

NetV.js 提供了基础的绘制节点-链接图的功能，用户可以通过简单的配置或导入数据集快速呈现图的形态。

# 基础图结构

给定节点链接数据，可以简单几行代码绘制出来。

# 内置数据集

NetV.js 内置了简单的数据集可供用户快速绘制一个复杂图结构。

# 大数据集

NetV.js 基于 WebGL 实现，可以支持百万级别元素绘制与交互。
